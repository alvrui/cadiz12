# Cádiz 1812 - Motor de Juego Narrativo-Político

Motor de juego narrativo basado en el contexto histórico de las Cortes de Cádiz de 1812.

## Estructura del Proyecto

- **src/config/** - Configuración de partidas, personajes, facciones y espacios
- **src/engine/** - Motor principal del juego con manejo de estados y eventos
- **src/sdk/** - Herramientas de generación de contenido
- **src/ui/** - Interfaz de terminal usando ratatui y crossterm
- **examples/** - Ejemplos de uso
- **configs/** - Configuraciones de partida predefinidas

## Requisitos

- Rust 2021 edition
- Dependencias: ver Cargo.toml

## Ejecución

```bash
# Ejecutar ejemplo de juego en terminal
cargo run --example terminal_game configs/basic/completa.json

# Verificar compilación
cargo check

# Ejecutar tests
cargo test
```
